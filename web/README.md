# Codex Ambassador Web

The Next.js web package for Codex Ambassador.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run lint
npm run build
```

## Vercel

Create a Vercel project from the repository and set its root directory to
`web`. Vercel detects the Next.js build settings automatically.

Keep non-secret settings in checked-in configuration files. Environment
variables are reserved for secrets and credentials.
